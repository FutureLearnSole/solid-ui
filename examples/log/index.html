The log module

On this page you'll find some examples from the log module. See API documentation for more information.

Status area: This is the playground which the examples will interact with. Click the various buttons to see their effect here.

Note that by default only msg, warn and error will add messages to the status area. You can change this by using setLevel.

log.msg: A simple message 


    






    

        
            log.warn: A warning message
        
    

    
    

    






    

        
            log.debug: A debug message
        
    

    
    

    






    

        
            log.info: An info message
        
    

    
    

    






    

        
            log.error: An error message
        
    

    
    

    






    

        
            log.success: A success message
        
    

    
    

    






    

        
            log.alert: An alert message
        
    

    

        This example uses alert to notify user of a message. It will not log in the status area unless
        window.alert is not available.
    

    
    

    






    

        
            log.clear: Clear messages
        
    

    

        Using this method you can clear the existing messages in the status area.
    

    
    

    






    

        
            log.setLevel: Configure which messages that are to be shown
        
    

    

        You can fine-tune which messages you want shown by adding together the corresponding numbers for each level:
    

    
    
        
  • Error: 1
    • 
        
  • Warning: 2
    • 
        
  • Message: 4
    • 
        
  • Success: 8
    • 
        
  • Info: 16
    • 
        
  • Debug: 32
    • 
    

    

        Setting level to 64 will show all messages.
    

    

        By default only error, warning, and normal messages will be shown, hence the number 7 that is shown by default
        in this example.
    

    
    

    






    

        
            log.dumpHTML: Dump all HTML in body in a log message
        
    

    
    

    






    

        
            log.logAscending: Start logging all messages in ascending order
        
    

    
    

    






    

        
            log.logDescending: Start logging all messages in descending order
